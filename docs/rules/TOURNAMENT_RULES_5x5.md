# Офіційні турнірні правила "Aether 5/5x5" (Neutral)

Цей документ описує суворі математичні та логічні правила для турнірного проведення гри у форматі 5х5 на 5 типів символів (Елементів) у режимі спільного управління (Neutral). Усі художні описи виключені для забезпечення однозначного трактування суддями.

---

## 1. Інвентар та Обладнання
1.1. **Дошка:** Квадратна сітка розміром 5х5 клітинок (загалом 25 полів).
1.2. **Топологія Дошки:** Дошка є тором. Це означає, що рух за межі лівого краю продовжується з правого краю (і навпаки), а рух за межі верхнього краю продовжується з нижнього (і навпаки).
1.3. **Фігури (Токени):** Загалом на старті є 25 базових токенів. Існує 5 унікальних типів маркувань (наприклад, цифри 1, 2, 3, 4, 5, або літери A, B, C, D, E). Кожного типу рівно по 5 штук. Фігури у грі не мають власника (ні білих, ні чорних).

---

## 2. Початкова розстановка (Setup)
2.1. Усі 25 клітинок дошки перед початком гри заповнені фігурами по одній у кожній клітинці.
2.2. Розподіл фігур відбувається строго за діагональним патерном, який описується математичною формулою: `Тип = (Рядок + Стовпець) mod 5`.
*Приклад розстановки (де 0, 1, 2, 3, 4 — типи фігур):*
* Рядок 0: `0, 1, 2, 3, 4`
* Рядок 1: `1, 2, 3, 4, 0`
* Рядок 2: `2, 3, 4, 0, 1`
* Рядок 3: `3, 4, 0, 1, 2`
* Рядок 4: `4, 0, 1, 2, 3`

---

## 3. Процес гри (Ходи)
3.1. Два гравці ходять по черзі. Гравець 1 (білі) робить перший хід. Пропускати хід **заборонено**.
3.2. Під час свого ходу гравець має право вибрати **будь-яку** існуючу на дошці фігуру і перемістити її.
3.3. **Рух:** Усі переміщення здійснюються за правилом "Ходу Коня" з класичних шахів (L-подібний стрибок: дві клітинки по горизонталі/вертикалі та одна перпендикулярно). З урахуванням топології тора (п. 1.2), кожна фігура завжди має рівно 8 доступних полів для приземлення.
3.4. **Переміщення на порожню клітинку:** Якщо фігура стрибає на клітинку, де немає іншої фігури, хід завершується. Фігура просто змінює своє положення.
3.5. **Злиття:** Якщо фігура стрибає на клітинку, де **вже стоїть** інша фігура, вони зливаються. 
* Фігура, що утворилася, набуває властивостей (маркувань) обох фігур (Математичне об'єднання множин).
* Якщо обидві фігури містили однаковий тип символу, він не дублюється. Рахується лише унікальна кількість символів. (Наприклад: фігура `[1, 2]` стрибає на фігуру `[2, 3]`. Результат — одна фігура на цій клітинці з властивостями `[1, 2, 3]`).

---

## 4. Умова Перемоги
4.1. **Миттєва перемога:** Гравець виграє партію тієї ж миті, як внаслідок його ходу на дошці утворюється фігура, яка містить **усі 5 доступних у грі маркувань** (`[1, 2, 3, 4, 5]`). У цей момент партія зупиняється, хід опонента не відбувається.

---

## 5. Додаткові турнірні правила
5.1. **Неможливість ходу:** Правила гарантують, що доки ніхто не виграв, на дошці є щонайменше 2 фігури, і у кожного гравця є законні ходи (злиття або переміщення в пусту клітинку). Ситуація Пату (Stalemate) математично неможлива.
5.2. **Правило 50 ходів:** Якщо протягом 50 послідовних ходів (по 25 ходів кожного гравця) відбувалися виключно переміщення фігур на пусті клітинки без ЖОДНОГО злиття (кількість фігур на дошці не зменшувалась), фіксується нічия за згодою.
5.3. **Доторк-хід (Touch-move):** Якщо гравець доторкнувся до фігури на спільній дошці та виконав валідний хід конем, відпустивши фігуру, хід вважається завершеним і не може бути скасований.
